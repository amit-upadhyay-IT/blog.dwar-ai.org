# Dwar-AI Blog

The technical blog of an engineer sharing learnings on the way to AI. 

🌐 **Live Site:** [https://blog.dwar-ai.org](https://blog.dwar-ai.org)

This project is built using [Astro](https://astro.build/) and the [Starlight](https://starlight.astro.build/) documentation framework, customized with the official blog plugin and a vintage theme.

## Local Development

To run this blog locally on your machine:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Open `http://localhost:4321` in your browser to see the site.
